#DatingSite

Проект представляет из себя веб-сайт для знакомств, в котором можно: 
1) Публиковать статьи на множество популярных тем
2) Ставить реакции на статьи, давая обратную связь
3) Добавлять людей в друзья, тем временем собирая клуб единомышленников
4) Получать достижения на сайте юудучи активным пользователем, выполняя задания
5) Возможность обратиться к поддержке

Стек: Python (Werkzeug, Flask (Flask-WTF, Flask_login, WTForms), Pymorphy3), SQLite3, HTML, CSS, JS.

DatingSite — это веб-платформа для знакомств и общения, ориентированная на поиск единомышленников через обмен контентом и социальное взаимодействие. Проект позволяет пользователям не только находить пары, но и формировать сообщества вокруг общих интересов.

🚀 Основные возможности:
- Публикация статей: Пользователи могут создавать и публиковать материалы на различные популярные темы.
- Система реакций: Возможность оставлять обратную связь на статьи с помощью реакций, что помогает оценивать популярность контента.
- Социальные связи: Функция добавления пользователей в друзья для формирования круга общения и клубов по интересам.
- Геймификация: Система достижений и заданий, мотивирующая пользователей к активности на платформе.
- (Виртуальная) поддержка пользователей: Встроенная возможность обращения в службу поддержки для решения технических или организационных вопросов.
  
🛠 Технологический стек
Проект разработан с использованием следующих технологий и библиотек:

Backend:
- Python
- Flask (микровеб-фреймворк):
1) Flask-WTF (интеграция WTForms для валидации форм)
2) Flask-Login (управление сессиями пользователей)
- Werkzeug (WSGI утилита)
- Pymorphy3 (морфологический анализ русского языка)
  
Database:
- SQLite3
  
Frontend:
- HTML5
- CSS3
- JavaScript
